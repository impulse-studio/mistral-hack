# Oxlint Configuration Editing

BEFORE editing any oxlint config, you MUST:

1. Read `.claude/docs/stacks/oxlint.md` for config reference
2. Use Context7 (`resolve-library-id` then `query-docs`) to verify any rule names, plugin names, or config syntax you are unsure about

Do not guess at rule names or plugin prefixes — verify them first.
