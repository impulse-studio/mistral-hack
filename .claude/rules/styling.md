# Styling & UI Components

BEFORE editing UI components or styles, you MUST:

1. Read `.claude/docs/stacks/styling.md` for the token system, shadcn conventions, and monorepo setup
2. Use Context7 (`resolve-library-id` then `query-docs`) to look up any shadcn/ui or Tailwind v4 syntax you are unsure about

Do not guess at token names or shadcn patterns — verify against the docs first.
