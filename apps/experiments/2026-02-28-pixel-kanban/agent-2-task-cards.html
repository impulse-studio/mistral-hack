10 CSS IMPLEMENTATION SPEC // pixel rules reference
PIXEL BORDERS
border: 3px solid
Inner: 1px inset glow
Hover: border-color swap
No border-radius ever
box-shadow: hard only
Drag: 4px 4px 0 0
TYPOGRAPHY
Headings: Press Start 2P
Labels: 6-7px pixel font
Body: 11px Inter
Code/IDs: JetBrains Mono
-webkit-font-smoothing: none
Sizes: multiples of 2px
ANIMATIONS
Use steps() not ease
Blink: steps(1) 0.6s
Pulse: steps(2) 1s
Bounce: steps(4) 0.8s
Sparkle: steps(3) 1.5s
Spin: steps(4) 2s
HP BAR THRESHOLDS
>50%: Green (healthy)
25-50%: Yellow (warn)
<25%: Red (danger)
100%: Full + done state
Segments: flex:1, 2px gap
Height: 10-12px
CARD ANATOMY
1. Priority stripe (4px)
2. Header (priority+ID)
3. Title (pixel font)
4. Description (body)
5. Due date (optional)
6. HP bar (optional)
7. Attachments (optional)
8. Labels
9. Footer (agent+XP)
INTERACTION STATES
Default: 3px border
Hover: border-hover color
Dragged: 4px shadow+rotate
Drop target: dashed blink
Selected: orange border
Done: 0.75 opacity
Epic: gold+sparkles+glow
